## UML
![](https://github.com/SpaceBirbMan/OOP/blob/main/Class/LRS_class.svg)
## ADT
### Данные Радиостанция
```
Название
Частота вещания
Общая информация
Статус вещания
Радиус покрытия вещания
Координаты местоположения
*Данные помещены в private поле (prv)*
```
### Операции Радиостанция
```
Конструктор
Вход: Название(inp), Частота(inp), Радиус(inp), Координаты(inp)
Значения по умолчанию:
Название(inp) - Untitled
Частота(inp) - 0.0
Радиус(inp) - 1.0
Координат(inp) - NULL

Процесс: 
Название(prv) = Название(inp)
Частота(prv) = Частота(inp)
Радиус(prv) = Радиус(inp)

Если Координаты = NULL то Координаты(prv) = [0.0 , 0.0]
Иначе Координаты(prv) = Координаты(inp)


Операция Задать название
Вход: Название
Предусловие: -
Процесс: Изменить название(prv) // prv - внутри объекта
Выход: -
Постусловие: -

Операция Задать частоту
Вход: Частота
Предусловие: Частота > 0
Процесс: Изменить частоту(prv) // prv - внутри объекта
Выход: -
Постусловие: -

Операция Задать информацию
Вход: Информация
Предусловие: 
Процесс: Изменить информацию(prv) // prv - внутри объекта
Выход: -
Постусловие: -

Операция Задать статус вещания
Вход: Статус
Предусловие: -
Процесс: Изменить статус(prv) // prv - внутри объекта
Выход: -
Постусловие: -

Операция Задать радиус
Вход: Радиус
Предусловие: Радиус > 0 
Процесс: Изменить радиус(prv)
Выход: -
Постусловие: -

Операция Задать координаты косвенно
Вход: Массив с координатами
Предусловие: -
Процесс: Изменить координаты(prv)
Выход: -
Постусловие: -

Операция Задать координаты прямо
Вход: Координаты
Предусловие: -
Процесс: Изменить координаты(prv)
Выход: -
Постусловие: -


Операция Получить название
Вход: -
Предусловие: -
Процесс: Получить название(prv)
Выход: Название
Постусловие: -

Операция Получить частоту
Вход: -
Предусловие: -
Процесс: Получить частоту(prv)
Выход: Частота
Постусловие: -

Операция Получить информацию
Вход: -
Предусловие: -
Процесс: Получить информацию(prv)
Выход: Информация
Постусловие: -

Операция Получить статус вещания
Вход: Статус
Предусловие: -
Процесс: Получить статус(prv)
Выход: Статус
Постусловие: -

Операция Получить радиус
Вход: - 
Предусловие: -
Процесс: Получить радиус(prv)
Выход: Радиус
Постусловие: -

Операция Получить координаты
Вход: - 
Предусловие: -
Процесс: Получить координаты(prv)
Выход: Координаты
Постусловие: - 

Операция Получить координаты в строку
Вход: - 
Предусловие: -
Процесс: Получить координаты(prv)
Выход: Координаты
Постусловие: -


Операция Переключить статус вещания
Вход: - 
Предусловие: -
Процесс: Если статус - ложь, то вых = истина иначе вых = ложь
Выход: вых
Постусловие: -

Операция Получить всю информацию
Вход: - 
Предусловие: -
Процесс:
Информация = название + частота + инфо + статус + радиус + координаты
Выход: Информация
Постусловие: -
```
