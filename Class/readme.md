## ADT
### Данные Радиостанция
```
Название sname
Частота вещания sfreq
Описание sdesc
Статус вещания sstat
Радиус покрытия вещания srad
Координаты местоположения scoord
*Данные помещены в private поле*
```
### Операции Радиостанция
```
Конструктор
Вход: Название, Частота, Радиус, Координаты
Значения по умолчанию:
Название - Untitled
Частота - 0.0
Радиус - 1.0
Координат - NULL

Процесс: 
sname = Название
sfreq = Частота
srad = Радиус

Если Координаты = NULL то scoord = [0.0 , 0.0]
Иначе scoord = Координаты


Операция Задать название
Вход: Название
Предусловие: -
Процесс: Изменить sname
Выход: -
Постусловие: -

Операция Задать частоту
Вход: Частота
Предусловие: Частота >= 0
Процесс: Изменить sfreq
Выход: -
Постусловие: -

Операция Задать информацию
Вход: Информация
Предусловие: 
Процесс: Изменить sdesc
Выход: -
Постусловие: -

Операция Задать статус вещания
Вход: Статус
Предусловие: -
Процесс: Изменить sstat
Выход: -
Постусловие: -

Операция Задать радиус
Вход: Радиус
Предусловие: Радиус >= 0 
Процесс: Изменить srad
Выход: -
Постусловие: -

Операция Задать координаты косвенно
Вход: Массив с координатами
Предусловие: -
Процесс: Изменить scoord
Выход: -
Постусловие: -

Операция Задать координаты прямо
Вход: Координаты
Предусловие: -
Процесс: Изменить scoord
Выход: -
Постусловие: -


Операция Получить название
Вход: -
Предусловие: -
Процесс: Получить sname
Выход: Название
Постусловие: -

Операция Получить частоту
Вход: -
Предусловие: -
Процесс: Получить sfreq
Выход: Частота
Постусловие: -

Операция Получить информацию
Вход: -
Предусловие: -
Процесс: Получить sdesc
Выход: Информация
Постусловие: -

Операция Получить статус вещания
Вход: Статус
Предусловие: -
Процесс: Получить sstat
Выход: Статус
Постусловие: -

Операция Получить радиус
Вход: - 
Предусловие: -
Процесс: Получить srad
Выход: Радиус
Постусловие: -

Операция Получить координаты
Вход: - 
Предусловие: -
Процесс: Получить scoord
Выход: Координаты
Постусловие: - 

Операция Получить координаты в строку
Вход: - 
Предусловие: -
Процесс: Получить scoord
Выход: Координаты
Постусловие: -


Операция Переключить статус вещания
Вход: - 
Предусловие: -
Процесс: Если статус - ложь, то вых = истина иначе вых = ложь
Выход: вых
Постусловие: -

Операция Получить всю информацию в строку
Вход: - 
Предусловие: -
Процесс:
Информация = название + частота + инфо + статус + радиус + координаты
Выход: Информация
Постусловие: -
```
